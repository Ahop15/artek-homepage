The process consists of the following steps:

1. The process begins with the Start phase.

2. After the Start phase, the process moves to the Needs Analysis phase.

3. Following the Needs Analysis phase, the process advances to the Feasibility Study phase.

4. After the Feasibility Study phase, the process proceeds to the Project Planning phase.

5. The Project Planning phase is followed by the Application Preparation phase.

6. After the Application Preparation phase, the process moves to the Ministry Application phase.

7. Following the Ministry Application phase, the process enters the Application Tracking phase.

8. During the Application Tracking phase, an Approval evaluation is performed.

9. If the Approval evaluation is positive (Yes), the process proceeds to the R&D / Design Center Setup phase.

10. If the Approval evaluation is negative (No), the process returns to the Revision phase.

11. After the Revision phase, the process returns to the Ministry Application phase.

12. Following the R&D / Design Center Setup phase, the process moves to the Operation Launch phase.

13. After the Operation Launch phase, the process advances to the Continuous Consulting phase.

14. Following the Continuous Consulting phase, the process reaches the Completed phase and the process is completed.
