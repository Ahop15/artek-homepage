Süreç şu adımlardan oluşur:

1. Süreç, Proje Fikri aşamasıyla başlar.
2. Proje Fikri aşamasından sonra Program Eşleştirme aşamasına geçilir.
3. Program Eşleştirme aşamasından sonra Proje Yazımı aşamasına geçilir.
4. Proje Yazımı aşamasından sonra Başvuru aşamasına geçilir.
5. Başvuru aşamasından sonra Sonuç değerlendirmesi yapılır.
6. Sonuç değerlendirmesi Kabul durumunda, Proje Yönetimi aşamasına geçilir.
7. Sonuç değerlendirmesi Red durumunda, Revizyon aşamasına geri dönülür.
8. Revizyon aşamasından sonra tekrar Proje Yazımı aşamasına geçilir.
9. Proje Yönetimi aşamasından sonra Raporlama aşamasına geçilir.
10. Raporlama aşamasından sonra Kapanış aşamasına ulaşılır ve süreç tamamlanır.
